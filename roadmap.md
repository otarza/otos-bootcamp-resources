# рЃЮрЃЌрЃЮрЃА рЃЉрЃБрЃЌрЃЦрЃћрЃЏрЃцрЃўрЃА рЃАрЃљрЃАрЃгрЃљрЃЋрЃџрЃЮ рЃњрЃћрЃњрЃЏрЃљ

1. рЃЎрЃЋрЃўрЃарЃљ #1 - рЃерЃћрЃАрЃљрЃЋрЃљрЃџрЃў рЃЋрЃћрЃЉ-рЃЊрЃћрЃЋрЃћрЃџрЃЮрЃърЃЏрЃћрЃюрЃбрЃерЃў;
2. рЃЎрЃЋрЃўрЃарЃљ #2 - рЃерЃћрЃАрЃљрЃЋрЃљрЃџрЃў рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃБрЃџ рЃўрЃюрЃЪрЃўрЃюрЃћрЃарЃўрЃљрЃерЃў;
3. рЃЎрЃЋрЃўрЃарЃљ #3 - HTML - рЃўрЃА рЃАрЃљрЃцрЃБрЃФрЃЋрЃџрЃћрЃЉрЃў;
4. рЃЎрЃЋрЃўрЃарЃљ #4 - CSS - рЃўрЃА рЃАрЃљрЃцрЃБрЃФрЃЋрЃџрЃћрЃЉрЃў рЃЊрЃљ Box Model;
5. рЃЎрЃЋрЃўрЃарЃљ #5 - рЃћрЃЊрЃўрЃбрЃЮрЃарЃћрЃЉрЃў, рЃЊрЃћрЃЋрЃћрЃџрЃЮрЃърЃЏрЃћрЃюрЃбрЃў рЃџрЃЮрЃЎрЃљрЃџрЃБрЃарЃљрЃЊ, рЃ░рЃЮрЃАрЃбрЃўрЃюрЃњрЃў рЃЊрЃљ рЃюрЃљрЃЏрЃБрЃерЃћрЃЋрЃарЃћрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃЦрЃЋрЃћрЃДрЃюрЃћрЃЉрЃўрЃА рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃ«рЃћрЃарЃ«рЃћрЃЉрЃў;
6. рЃЎрЃЋрЃўрЃарЃљ #6 - CSS - рЃАрЃљрЃБрЃЎрЃћрЃЌрЃћрЃАрЃЮ рЃърЃарЃљрЃЦрЃбрЃўрЃЎрЃћрЃЉрЃў рЃЊрЃљ CSS рЃљрЃарЃЦрЃўрЃбрЃћрЃЦрЃбрЃБрЃарЃўрЃА рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃЏрЃўрЃЊрЃњрЃЮрЃЏрЃћрЃЉрЃў;
7. рЃЎрЃЋрЃўрЃарЃљ #7 - Responsive: Media Queries, Grids, Flexbox рЃЊрЃљ рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃАрЃў рЃърЃарЃЮрЃћрЃЦрЃбрЃћрЃЉрЃў;
8. рЃЎрЃЋрЃўрЃарЃљ #8 - рЃерЃћрЃАрЃљрЃЋрЃљрЃџрЃў JavaScript - рЃерЃў: рЃўрЃюрЃбрЃарЃЮ, рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃў, рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃў, рЃЏрЃљрЃАрЃўрЃЋрЃћрЃЉрЃў, рЃърЃўрЃарЃЮрЃЉрЃўрЃЌрЃў рЃЮрЃърЃћрЃарЃљрЃбрЃЮрЃарЃћрЃЉрЃў, рЃАрЃЎрЃЮрЃБрЃърЃў, рЃфрЃўрЃЎрЃџрЃћрЃЉрЃў, рЃЊрЃћрЃЉрЃљрЃњрЃўрЃюрЃњрЃў рЃЊрЃљ рЃЎрЃўрЃЊрЃћрЃЋ рЃЉрЃћрЃЋрЃарЃў рЃарЃљрЃЏ
9. рЃЎрЃЋрЃўрЃарЃљ #9 - рЃерЃћрЃАрЃљрЃЋрЃљрЃџрЃў Git - рЃАрЃљ рЃЊрЃљ source control рЃАрЃўрЃАрЃбрЃћрЃЏрЃћрЃЉрЃерЃў;
10. рЃЎрЃЋрЃўрЃарЃљ #10 - рЃерЃћрЃАрЃљрЃЋрЃљрЃџрЃў рЃЉрЃарЃЮрЃБрЃќрЃћрЃарЃўрЃА рЃўрЃюрЃбрЃћрЃарЃЦрЃфрЃўрЃљрЃерЃў: HTML рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃўрЃА рЃЏрЃљрЃюрЃўрЃърЃБрЃџрЃљрЃфрЃўрЃљ, DOM, рЃцрЃЮрЃарЃЏрЃћрЃЉрЃў рЃЊрЃљ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃЌрЃљрЃю рЃБрЃарЃЌрЃўрЃћрЃарЃЌрЃЮрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА рЃАрЃљрЃГрЃўрЃарЃЮ рЃЏрЃћрЃЦрЃљрЃюрЃўрЃќрЃЏрЃћрЃЉрЃўрЃА рЃњрЃљрЃфрЃюрЃЮрЃЉрЃљ;
11. рЃЎрЃЋрЃўрЃарЃљ #11 - рЃерЃћрЃАрЃљрЃЋрЃљрЃџрЃў Javascript - рЃерЃў - рЃюрЃљрЃгрЃўрЃџрЃў II: рЃЎрЃџрЃљрЃАрЃћрЃЉрЃў, рЃЏрЃЮрЃЊрЃБрЃџрЃћрЃЉрЃў, рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљ;
12. рЃЎрЃЋрЃўрЃарЃљ #12 - рЃерЃћрЃАрЃљрЃЋрЃљрЃџрЃў рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃљрЃерЃў: TDD рЃЊрЃљ рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃАрЃў рЃўрЃљрЃарЃљрЃдрЃћрЃЉрЃўрЃА рЃњрЃљрЃфрЃюрЃЮрЃЉрЃљ/рЃљрЃЌрЃЋрЃўрЃАрЃћрЃЉрЃљ;
13. рЃЎрЃЋрЃўрЃарЃљ #13 - HTTP & JS: HTTP - рЃўрЃА рЃАрЃљрЃцрЃБрЃФрЃЋрЃџрЃћрЃЉрЃў рЃЊрЃљ JavaScript - рЃерЃў рЃЏрЃўрЃАрЃў рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃА рЃњрЃќрЃћрЃЉрЃў, рЃљрЃАрЃўрЃюрЃЦрЃарЃЮрЃюрЃБрЃџрЃЮрЃЉрЃљ рЃЊрЃљ рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃАрЃў рЃўрЃљрЃарЃљрЃдрЃћрЃЉрЃў;
14. рЃЎрЃЋрЃўрЃарЃљ #14 - рЃерЃћрЃАрЃљрЃЋрЃљрЃџрЃў React - рЃерЃў: рЃЋрЃўрЃарЃбрЃБрЃљрЃџрЃБрЃарЃў DOM, рЃЌрЃћрЃЏрЃцрЃџрЃћрЃўрЃЌрЃћрЃЉрЃў, рЃЎрЃЮрЃЏрЃърЃЮрЃюрЃћрЃюрЃбрЃћрЃЉрЃў рЃЊрЃљ JSX, React рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃћрЃЉрЃўрЃА рЃЊрЃћрЃцрЃџрЃЮрЃўрЃЏрЃћрЃюрЃЌрЃў рЃАрЃћрЃарЃЋрЃћрЃарЃќрЃћ;
15. рЃЎрЃЋрЃўрЃарЃљ #15 - React - рЃюрЃљрЃгрЃўрЃџрЃў II: рЃўрЃюрЃбрЃћрЃарЃљрЃЦрЃфрЃўрЃљ рЃЎрЃЮрЃЏрЃърЃЮрЃюрЃћрЃюрЃбрЃћрЃЉрЃА рЃерЃЮрЃарЃўрЃА, рЃ░рЃБрЃЎрЃћрЃЉрЃў, рЃарЃЮрЃБрЃбрЃћрЃарЃў, рЃЎрЃЮрЃЏрЃърЃЮрЃюрЃћрЃюрЃбрЃћрЃЉрЃўрЃА рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃљ, Storybook - рЃћрЃЉрЃў;
16. рЃЎрЃЋрЃўрЃарЃљ #16 - Git - рЃюрЃљрЃгрЃўрЃџрЃў II: рЃЉрЃарЃћрЃюрЃЕрЃўрЃюрЃњрЃў, рЃЏрЃћрЃюрЃћрЃ»рЃЏрЃћрЃюрЃбрЃў, рЃЎрЃЮрЃџрЃљрЃЉрЃЮрЃарЃљрЃфрЃўрЃљ;
17. рЃЎрЃЋрЃўрЃарЃљ #17 - рЃерЃћрЃАрЃљрЃЋрЃљрЃџрЃў Backend рЃЊрЃћрЃЋрЃћрЃџрЃЮрЃърЃЏрЃћрЃюрЃбрЃерЃў: рЃарЃљ рЃљрЃарЃўрЃА рЃЉрЃћрЃЦрЃћрЃюрЃЊ рЃЊрЃћрЃЋрЃћрЃџрЃЮрЃърЃЏрЃћрЃюрЃбрЃў, Node.js - рЃўрЃА рЃАрЃљрЃгрЃДрЃўрЃАрЃћрЃЉрЃў, рЃЏрЃЮрЃЊрЃБрЃџрЃћрЃЉрЃўрЃА рЃЏрЃћрЃюрЃћрЃ»рЃЏрЃћрЃюрЃбрЃў, Express.js, рЃарЃЮрЃБрЃбрЃћрЃЉрЃў, рЃерЃћрЃЦрЃЏрЃћрЃюрЃў рЃАрЃљрЃЎрЃБрЃЌрЃљрЃарЃў рЃЉрЃћрЃЦрЃћрЃюрЃЊ API;
18. рЃЎрЃЋрЃўрЃарЃљ #18 - рЃљрЃЋрЃбрЃЮрЃарЃўрЃќрЃљрЃфрЃўрЃљ/рЃарЃћрЃњрЃўрЃАрЃбрЃарЃљрЃфрЃўрЃљ, рЃАрЃћрЃАрЃўрЃћрЃЉрЃў, рЃърЃљрЃарЃЮрЃџрЃўрЃЌ рЃљрЃЋрЃбрЃЮрЃарЃўрЃќрЃљрЃфрЃўрЃљ, Oauth 2.0, 
19. рЃЎрЃЋрЃўрЃарЃљ #19 - рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃљ - рЃюрЃљрЃгрЃўрЃџрЃў II: рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃљ Backend - рЃќрЃћ;
20. рЃЎрЃЋрЃўрЃарЃљ #20 - рЃерЃћрЃАрЃљрЃЋрЃљрЃџрЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЉрЃљрЃќрЃћрЃЉрЃерЃў: рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЉрЃљрЃќрЃЉрЃўрЃА рЃбрЃўрЃърЃћрЃЉрЃў, рЃерЃћрЃАрЃљрЃЋрЃљрЃџрЃў PostgreSQL - рЃерЃў, рЃарЃћрЃџрЃљрЃфрЃўрЃБрЃарЃў рЃЉрЃљрЃќрЃћрЃЉрЃўрЃА рЃЊрЃўрЃќрЃљрЃўрЃюрЃўрЃА рЃАрЃљрЃцрЃБрЃФрЃЋрЃџрЃћрЃЉрЃў, рЃюрЃЮрЃарЃЏрЃљрЃџрЃўрЃќрЃљрЃфрЃўрЃљ;
21. рЃЎрЃЋрЃўрЃарЃљ #21 - рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЉрЃљрЃќрЃћрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ рЃЉрЃћрЃЦрЃћрЃюрЃЊ рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃћрЃЉрЃерЃў Node.js - рЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ, ORM - рЃћрЃЉрЃў;
22. рЃЎрЃЋрЃўрЃарЃљ #22 - рЃерЃћрЃАрЃљрЃЋрЃљрЃџрЃў API рЃЊрЃўрЃќрЃљрЃўрЃюрЃАрЃљ рЃЊрЃљ рЃЊрЃћрЃЋрЃћрЃџрЃЮрЃърЃЏрЃћрЃюрЃбрЃерЃў: Swagger рЃЊрЃљ OpenAPI;
23. рЃЎрЃЋрЃўрЃарЃљ #23 - рЃцрЃарЃЮрЃюрЃбрЃўрЃАрЃљ рЃЊрЃљ рЃЉрЃћрЃЦрЃўрЃА рЃЊрЃљрЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃљ;
24. рЃЎрЃЋрЃўрЃарЃљ #24 - рЃЋрЃћрЃЉрЃўрЃА рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮрЃћрЃЉрЃўрЃА рЃцрЃБрЃюрЃЊрЃљрЃЏрЃћрЃюрЃбрЃБрЃарЃў рЃАрЃљрЃЎрЃўрЃЌрЃ«рЃћрЃЉрЃўрЃА рЃњрЃљрЃюрЃ«рЃўрЃџрЃЋрЃљ;
25. рЃЎрЃЋрЃўрЃарЃљ #25 - рЃерЃћрЃАрЃљрЃЋрЃљрЃџрЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃћрЃЉрЃерЃў;
26. рЃЎрЃЋрЃўрЃарЃљ #26 - рЃерЃћрЃАрЃљрЃЋрЃљрЃџрЃў рЃљрЃџрЃњрЃЮрЃарЃўрЃЌрЃЏрЃћрЃЉрЃерЃў;
27. рЃЎрЃЋрЃўрЃарЃљ #27 - рЃерЃћрЃЊрЃљрЃарЃћрЃЉрЃўрЃЌ рЃарЃЌрЃБрЃџрЃў рЃЊрЃљ рЃфрЃюрЃЮрЃЉрЃўрЃџрЃў рЃљрЃџрЃњрЃЮрЃарЃўрЃЌрЃЏрЃћрЃЉрЃў, рЃАрЃбрЃљрЃюрЃЊрЃљрЃарЃбрЃБрЃџрЃў рЃФрЃћрЃЉрЃюрЃўрЃА рЃљрЃџрЃњрЃЮрЃарЃўрЃЌрЃЏрЃћрЃЉрЃў;
28. рЃЎрЃЋрЃўрЃарЃљ #28 - рЃЏрЃЮрЃЏрЃќрЃљрЃЊрЃћрЃЉрЃљ рЃўрЃюрЃбрЃћрЃарЃЋрЃўрЃБрЃАрЃЌрЃЋрЃўрЃА: рЃарЃћрЃќрЃўрЃБрЃЏрЃћрЃА рЃЏрЃЮрЃЏрЃќрЃљрЃЊрЃћрЃЉрЃљ, рЃюрЃљрЃАрЃгрЃљрЃЋрЃџрЃў рЃАрЃљрЃЎрЃўрЃЌрЃ«рЃћрЃЉрЃўрЃА рЃњрЃљрЃЊрЃљрЃ«рЃћрЃЊрЃЋрЃљ рЃЊрЃљ рЃЊрЃўрЃЊрЃў рЃАрЃБрЃарЃљрЃЌрЃўрЃА рЃЊрЃљрЃюрЃљрЃ«рЃЋрЃљ, рЃАрЃљрЃЎрЃБрЃЌрЃљрЃарЃў рЃўрЃАрЃбрЃЮрЃарЃўрЃўрЃА рЃЏрЃЮрЃцрЃўрЃЦрЃарЃћрЃЉрЃљ, рЃЊрЃљрЃгрЃћрЃарЃљ рЃЊрЃљ рЃЋрЃљрЃарЃ»рЃўрЃерЃў рЃЏрЃўрЃА рЃЏрЃЮрЃДрЃЮрЃџрЃљрЃерЃў. рЃЋрЃљрЃарЃ»рЃўрЃерЃў рЃАрЃљрЃЎрЃБрЃЌрЃљрЃарЃў рЃЌрЃљрЃЋрЃўрЃА, рЃБрЃюрЃљрЃарЃћрЃЉрЃўрЃА рЃЊрЃљ рЃюрЃљрЃЏрЃБрЃерЃћрЃЋрЃарЃћрЃЉрЃўрЃА рЃърЃарЃћрЃќрЃћрЃюрЃбрЃљрЃфрЃўрЃљрЃерЃў;
29. рЃЎрЃЋрЃўрЃарЃљ #29 - "рЃЏрЃЮрЃЎ рЃўрЃюрЃбрЃћрЃарЃЋрЃўрЃБрЃћрЃЉрЃўрЃА" рЃЎрЃЋрЃўрЃарЃљ - рЃўрЃюрЃбрЃћрЃарЃЋрЃўрЃБрЃА рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃАрЃў рЃњрЃљрЃарЃћрЃЏрЃЮрЃА рЃЏрЃЮрЃгрЃДрЃЮрЃЉрЃљ рЃАрЃљрЃфрЃЊрЃћрЃџрЃў рЃўрЃюрЃбрЃћрЃарЃЋрЃўрЃБрЃћрЃЉрЃўрЃА рЃЕрЃљрЃбрЃљрЃарЃћрЃЉрЃљ рЃарЃЮрЃњрЃЮрЃарЃф рЃ»рЃњрЃБрЃцрЃўрЃА рЃАрЃ«рЃЋрЃљ рЃгрЃћрЃЋрЃарЃћрЃЉрЃЌрЃљрЃю рЃљрЃАрЃћрЃЋрЃћ рЃЏрЃћрЃюрЃбрЃЮрЃарЃћрЃЉрЃЌрЃљрЃю;  
30. рЃЎрЃЋрЃўрЃарЃљ #30 - рЃАрЃљрЃЏрЃБрЃерЃљрЃЮрЃА рЃФрЃћрЃЉрЃюрЃўрЃА рЃЊрЃљрЃгрЃДрЃћрЃЉрЃљ, рЃЋрЃљрЃЎрЃљрЃюрЃАрЃўрЃћрЃЉрЃўрЃА рЃЏрЃЮрЃФрЃўрЃћрЃЉрЃљ, рЃарЃћрЃЎрЃЮрЃЏрЃћрЃюрЃЊрЃљрЃфрЃўрЃћрЃЉрЃўрЃА рЃЏрЃўрЃдрЃћрЃЉрЃљ, рЃЎрЃЮрЃЏрЃърЃљрЃюрЃўрЃћрЃЉрЃЌрЃљрЃю рЃЎрЃЮрЃЏрЃБрЃюрЃўрЃЎрЃљрЃфрЃўрЃўрЃА рЃЊрЃљрЃгрЃДрЃћрЃЉрЃљ

30 - рЃћ рЃЎрЃЋрЃўрЃарЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ рЃърЃарЃЮрЃфрЃћрЃАрЃў рЃњрЃарЃФрЃћрЃџрЃЊрЃћрЃЉрЃљ рЃЊрЃљ рЃЎрЃЮрЃЏрЃБрЃюрЃўрЃЎрЃљрЃфрЃўрЃљрЃА рЃЋрЃўрЃюрЃљрЃарЃЕрЃБрЃюрЃћрЃЉрЃЌ рЃЏрЃљрЃюрЃљрЃЏ рЃАрЃљрЃюрЃљрЃЏ рЃ»рЃњрЃБрЃцрЃўрЃА рЃДрЃЋрЃћрЃџрЃљ рЃгрЃћрЃЋрЃарЃў рЃљрЃа рЃЊрЃљрЃўрЃгрЃДрЃћрЃЉрЃА рЃЏрЃБрЃерЃљрЃЮрЃЉрЃљрЃА. рЃЋрЃљрЃњрЃарЃФрЃћрЃџрЃћрЃЉрЃЌ рЃЋрЃљрЃЎрЃљрЃюрЃАрЃўрЃћрЃЉрЃўрЃА рЃФрЃћрЃЉрЃюрЃљрЃА, рЃарЃћрЃЎрЃЮрЃЏрЃћрЃюрЃЊрЃљрЃфрЃўрЃћрЃЉрЃўрЃА рЃњрЃљрЃгрЃћрЃЋрЃљрЃА рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃЎрЃЮрЃЏрЃърЃљрЃюрЃўрЃћрЃЉрЃерЃў рЃЊрЃљ рЃћрЃарЃЌрЃЏрЃљрЃюрЃћрЃЌрЃўрЃА рЃЊрЃљрЃ«рЃЏрЃљрЃарЃћрЃЉрЃљрЃА рЃарЃЮрЃЏ рЃДрЃЋрЃћрЃџрЃљрЃЏ рЃЏрЃљрЃЦрЃАрЃўрЃЏрЃљрЃџрЃБрЃарЃљрЃЊ рЃАрЃгрЃарЃљрЃцрЃљрЃЊ рЃЊрЃљ рЃЏрЃљрЃЦрЃАрЃўрЃЏрЃљрЃџрЃБрЃарЃљрЃЊ рЃЎрЃљрЃарЃњрЃў рЃърЃўрЃарЃЮрЃЉрЃћрЃЉрЃўрЃЌ рЃЊрЃљрЃўрЃгрЃДрЃЮрЃА рЃЏрЃБрЃерЃљрЃЮрЃЉрЃљ.
